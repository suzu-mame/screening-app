# 文献スクリーニングアプリ

タイトル・抄録スクリーニングをブラウザだけで行える単体HTMLアプリ。PubMedの `.nbib` / CSV を自分で読み込んで、Include / Maybe / Exclude を記録する。**データは端末内のブラウザだけで処理され、外部に送信されない**（サーバなし・オフライン動作）。

- 本体: **`index.html`**（これがアプリ。ローカルでもこれを開く）
- 使い方: `使い方とスマホ設定.md`

## 主な機能
- `.nbib`（MEDLINE形式）/ CSV の読み込み（ブラウザ内でパース、DOI等も抽出）
- 任意で日本語訳JSON・判定続きCSVの読み込み
- 設定パネルで **包含・除外基準と除外ラベルを入力**（📋基準で表示、Exclude理由に連動）
- 自動フィルタ: ① 言語(LA) ② 出版タイプ(PT) ③ 発行年レンジ による自動除外
- 書き出し: 全判定ログ / フルテキスト用 / 集計サマリー（除外理由別件数）/ 3シートExcel
- PC↔スマホ引き継ぎ（全判定ログCSVを読み込んで判定を復元）

## 使い方（ローカル）
`index.html` をブラウザで開くだけ。スマホで使う場合は GitHub Pages 等でURL化すると快適（`GitHub管理手順.md` 参照）。

## データの扱い
読み込んだ文献データも判定結果も、すべて利用者の端末内（ブラウザのlocalStorage）に保存され、どこにも送信されない。PubMedの書誌は公開情報。

## ライセンス / 引用
